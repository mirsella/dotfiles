# dotfiles

Personal dotfiles managed with [chezmoi](https://www.chezmoi.io/).

This repo contains shell, terminal, tmux, SSH, and desktop configuration used on my machines.

## Apply on a machine

```bash
chezmoi init --apply mirsella/dotfiles
```

## If the repo is already cloned

```bash
chezmoi apply
```

This is a personal setup, so review the files before applying everything as-is.
